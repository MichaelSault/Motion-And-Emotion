Animating (Multiple) Images Along a Path!

Want to make an animated path like this one?


First you must learn the controls:

Press Q and E to rotate in either direction.

*Pressing a second time will stop the animation*


Press W and S to stretch and shrink vertically.

*Pressing a second time will stop the animation*


Press D and A to widen and thin horizontally.

*Pressing a second time will stop the animation*

Now we can apply these to path recording:

Make sure to only animate the image when you are drawing the path.

You can use the current image as a reference for what the image will look like at that point in the path.


When setting the path to an image, make sure to first set the image, and then select the path you wish to apply to it.


You should now be able to record a video with multiple animated paths!
